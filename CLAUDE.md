# Project: Modular Patch Docs

Eurorack modular synth patch documentation tool built with React + Vite + TypeScript + Tailwind.

## Dev Commands
- `npm run dev` — start dev server
- `npx vitest run` — run tests
- `npx tsc -b --noEmit` — type check

## Rules
- **Always browser-test before saying you're done.** Start the dev server and verify the change works in Chrome using the browser automation tools. No exceptions.
